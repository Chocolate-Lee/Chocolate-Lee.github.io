在UIScrollView上增加滑动手势（Pan）

view上有一个scrollView，现在想在view上加一个Pan手势，需求是：当向下划的时候，整个view动，但是scrollView不动；其它情况下scrollView动而view不动。

解决方法：

-(BOOL)gestureRecognizer:(UIGestureRecognizer *)gestureRecognizer shouldRecognizeSimultaneouslyWithGestureRecognizer:(UIGestureRecognizer *)otherGestureRecognizer {
  return YES;
}

// 给加的手势设置代理, 并实现此协议方法
- (BOOL)gestureRecognizerShouldBegin:(UIGestureRecognizer *)gestureRecognizer {
  if ([gestureRecognizer isKindOfClass:[UIPanGestureRecognizer class]]) {

    UIPanGestureRecognizer *pan = (UIPanGestureRecognizer *)gestureRecognizer;
    CGPoint pos = [pan velocityInView:pan.view];
    if (pos.y > 0) {
      self.contentScrollView.scrollEnabled = NO;
      return YES;
    }
  }

  return NO;
}

Linux下Maven命令创建多模块Web项目

用mvn命令快捷地创建多模块的java web工程，再倒入eclipse。

1.首先，创建一个父项目：

mvn archetype:generate -DgroupId=com.lsan -DartifactId=helloweb

选模板时可以选用默认的。

2.创建子模块

工程创建后，修改pom.xml，将打包类型由jar修改成pom。如果不修改这个参数，后面将无法创建子模块。然后在这个文件夹里创建需要的普通maven模块，命令如下：

mvn archetype:generate -DgroupId=com.lsan -DartifactId=helloweb-core 

mvn archetype:generate -DgroupId=com.lsan -DartifactId=helloweb-entity 

mvn archetype:generate -DgroupId=com.lsan -DartifactId=helloweb-parent

3.创建web模块：

mvn archetype:generate -DgroupId=com.lsan -DartifactId=helloweb-web

在选取模板时，先输入web进行过滤，再选取想要的web模板。
