SpringMVC（一）

  • Java
  • Maven
Java

一、准备

1.jdk
2.tomcat
3.maven

二、eclipse配置

1.配置maven
a).在Preferences -> Maven -> Installations 中 Add一个maven的路径，选中并应用。

b).在Preferences -> Maven -> User Settings 中 配置maven目录下conf中的settings.xml
，选中并应用。


2.配置tomcat
a).控制台中，点击Servers，点击提示”No servers are available, Click this link to create a new server…”添加Server。或者新建一个Server。
b).选中Apache下对应的tomcat，点击Next。
c).选择tomcat的路径，并选择相应的jre版本。


三、创建工程

1.新建一个maven project



2.修改pom文件，并刷新maven
3.在java源文件夹下创建service，dao，entity，controller，common包
4.在java源文件下创建resources源文件夹，在里面添加配置文件
5.在webapp下创建css，images，javascript，WEB-INF文件夹。
6.在WEB-INF下创建page文件夹及配置文件。
7.在page下创建jsp文件。
8.部署，并启动tomcat。
