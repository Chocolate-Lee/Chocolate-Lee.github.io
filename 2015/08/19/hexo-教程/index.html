Hexo

既然用了hexo，就先写篇教程吧，各类问题肯定会遇到不少。

一、搭建环境（Linux）

1.首先下载node.js。选下载一个版本，我选的是最新的。
2.解压，编译、安装：

./configure --prefix=/usr/local/node
make 
make install

3.安装npm（如果没安装过这个指令的话）

curl -L https://npmjs.org/install.sh | sh

PS：可能会提示没有安装node.js，找不到node，原因是他会默认去/usr/local/bin/node去找，但是我的安装路径是自己定义的，可以建立一个软链接。

ln -s /mnt/node/bin/node /user/local/bin/node

再执行上条安装语句。
4.安装hexo

npm install -g hexo-cli

初始化

hexo init <folder>

如果指定 folder，会在目前的文件夹下建立一个名为 的文件夹；否则会在目前文件夹初始化。

5.安装依赖包

npm install

至此，hexo搭建完毕。

二、基础配置

这个可以说很烦，刚接触的时候也找不到个好点的教程。在此我也只就自己遇到的问题列出一些解决办法。

1.hexo的全局配置文件是根目录下的_config.xml

2.更换主题

先下载主题

git clone https://github.com/litten/hexo-theme-yilia.git

将主题放置到根目录的themes下，可以进入每个主题下使用git指令更新

3.设置主题，主题的配置文件是每个主题的文件夹内的_config.xml

三、发布文章

hexo new "postName" #新建文章
hexo new page "pageName" #新建页面
hexo generate #生成静态页面至public目录
hexo server #开启预览访问端口（默认端口4000，'ctrl + c'关闭server）
hexo deploy #将.deploy目录部署到GitHub

修改文章在根目录下的source/_posts/内，每个md文件就是文章，可以进行修改。
