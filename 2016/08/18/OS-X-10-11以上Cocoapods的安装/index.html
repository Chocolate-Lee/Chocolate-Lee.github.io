OS X 10.11以上Cocoapods的安装

  • Cocoapods
Cocoapods

一、安装Xcode

二、安装 RVM

1.安装RVM

curl -L https://get.rvm.io | bash -s stable

2.载入RVM环境

source ~/.rvm/scripts/rvm

3.检查一下是否安装正确

rvm -v

rvm 1.22.17 (stable) by Wayne E. Seguin wayneeseguin@gmail.com, Michal Papis mpapis@gmail.com [https://rvm.io/]

三、用 RVM 安装 Ruby 环境

1.列出已知的ruby版本

rvm list known

2.可以选择现有的rvm版本来进行安装（10.11必须为rvm 2.2.0以上）

rvm install 2.3.0

3.查询已经安装的ruby

rvm list

4.卸载一个已安装版本 

rvm remove 1.9.2

四、设置 Ruby 版本

1.RVM 装好以后，需要执行下面的命令将指定版本的 Ruby 设置为系统默认版本

rvm 2.3.0 --default

同样，也可以用其他版本号，前提是你有用 rvm install 安装过那个版本

2.这个时候你可以测试是否正确

ruby -v

ruby 2.0.0p247 (2013-06-27 revision 41674) [x86_64-darwin13.0.0]

gem -v

2.1.6

五、替换镜像

1.安装CocoaPods时我们要访问cocoapods.org，用淘宝的RubyGems镜像来代替官方版本，执行以下命令

gem sources --remove https://rubygems.org/
gem sources -a https://ruby.taobao.org/

2.为了验证你的Ruby镜像是并且仅是taobao，可以用以下命令查看：

gem sources -l

只有在终端中出现下面文字才表明你上面的命令是成功的：
*** CURRENT SOURCES ***
http://ruby.taobao.org/

六、安装Cocoapods

1.1..安装Cocoapods,在终端输入命令

sudo gem install -n /usr/local/bin cocoapods
sudo xcode-select --switch /Applications/Xcode.app
pod setup

它需要一点时间来完成，你等就是了。如果安装失败 ~/.cocoapods 里面是空的，就需要重新setup,命令如下：

pod repo remove master  
pod setup
